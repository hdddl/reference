Signalling System No.7 备忘清单
===

SS7 既 Signalling System No.7 是一种被广泛应用在公共交换电话网、蜂窝通信网络等现代通信网络的共路信令系统。七号信令系统是国际电信联盟推荐首选的标准信令系统。
